# **프로젝트 목적 및 이유**  
최근 케이팝 데몬 헌터즈로 인해 한국풍에 관한 사람들의 관심이 늘어났으나 정작 신작 게임에는 한국풍보다는 기존에 보증된류의 게임만 계속 생산하여 유저들의 불만을 야기하고 있다. 새로운 도전을 하지 않고 계속 비슷한 게임만 나오는 탓에 유저들은 기존에 하던 게임을 버리고 새로운 게임을 해야 할 의미를 느끼지 못하고 있다.또한 SCP같은 유명 괴담류 게임 또한 인기를 얻고 있으나, 한국의 설화를 이용한 게임은 없어 유저들은 아쉬움을 느끼고 있다.
     이를 해결하기 위해서 한국 설화중 가장 인지도가 높은 산군을 기반으로 ‘한국풍 호러 카드턴제RPG 게임’을 만들기로 결정하였다.이 게임의 목적은 유저들의 요구를 만족하면서 한국풍 호러가 수많은 유저들에게 어필 될 수 있게 한다.단순한 호러게임이 아닌 전략이 필요한 카드 RPG를 결합함으로서 유저들의 흥미도를 높이고, 플레이 시간을 늘리는 효과를 기대한다.
